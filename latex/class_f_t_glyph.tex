\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_f_t_glyph}{FTGlyph}} is the base class for FTGL glyphs.

It provides the interface between Freetype glyphs and their open\+GL renderable counterparts. This is an abstract class and derived classes must implement the {\ttfamily render} function.
