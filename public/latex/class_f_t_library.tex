\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_f_t_library}{FTLibrary}} class is the global accessor for the Freetype library.

This class encapsulates the Freetype Library. This is a singleton class and ensures that only one FT\+\_\+\+Library is in existence at any one time. All constructors are private therefore clients cannot create or instantiate this class themselves and must access it\textquotesingle{}s methods via the static {\ttfamily \mbox{\hyperlink{class_f_t_library_ab57f88d6eddafcaf2ae1a518e8455ebd}{FTLibrary\+::\+Instance()}}} function.

Just because this class returns a valid {\ttfamily \mbox{\hyperlink{class_f_t_library}{FTLibrary}}} object doesn\textquotesingle{}t mean that the Freetype Library has been successfully initialised. Clients should check for errors. You can initialse the library AND check for errors using the following code... {\ttfamily err = \mbox{\hyperlink{class_f_t_library_ab57f88d6eddafcaf2ae1a518e8455ebd}{FTLibrary\+::\+Instance()}}.Error();}
