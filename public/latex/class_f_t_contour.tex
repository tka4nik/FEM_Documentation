\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_f_t_contour}{FTContour}} class is a container of points that describe a vector font outline. It is used as a container for the output of the bezier curve evaluator in \mbox{\hyperlink{class_f_t_vectoriser}{FTVectoriser}}.
