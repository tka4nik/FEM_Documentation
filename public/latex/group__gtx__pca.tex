\doxysubsection{Detailed Description}
Include \texorpdfstring{$<$}{<}\mbox{\hyperlink{pca_8hpp}{glm/gtx/pca.\+hpp}}\texorpdfstring{$>$}{>} to use the features of this extension.

Implements functions required for fundamental \textquotesingle{}princple component analysis\textquotesingle{} in 2D, 3D, and 4D\+: 1) Computing a covariance matrics from a list of {\itshape relative} position vectors 2) Compute the eigenvalues and eigenvectors of the covariance matrics This is useful, e.\+g., to compute an object-\/aligned bounding box from vertices of an object. \href{https://en.wikipedia.org/wiki/Principal_component_analysis}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Principal\+\_\+component\+\_\+analysis}}

Example\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{std::vector<glm::dvec3> ptData;}
\DoxyCodeLine{\textcolor{comment}{// ... fill ptData with some point data, e.g. vertices}}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{structglm_1_1vec}{glm::dvec3}} center = computeCenter(ptData);}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{structglm_1_1mat}{glm::dmat3}} covarMat = \mbox{\hyperlink{group__gtx__pca_ga4290358424d0cb9bcdfc5106f1db557e}{glm::computeCovarianceMatrix}}(ptData.data(), ptData.size(), center);}
\DoxyCodeLine{}
\DoxyCodeLine{\mbox{\hyperlink{structglm_1_1vec}{glm::dvec3}} evals;}
\DoxyCodeLine{\mbox{\hyperlink{structglm_1_1mat}{glm::dmat3}} evecs;}
\DoxyCodeLine{\textcolor{keywordtype}{int} evcnt = \mbox{\hyperlink{group__gtx__pca_ga0586007af1073c8b0f629bca0ee7c46c}{glm::findEigenvaluesSymReal}}(covarMat, evals, evecs);}
\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{keywordflow}{if}(evcnt != 3)}
\DoxyCodeLine{    \textcolor{comment}{// ... error handling}}
\DoxyCodeLine{}

\DoxyCodeLine{}
\DoxyCodeLine{\textcolor{comment}{// ... now evecs[0] points in the direction (symmetric) of the largest spatial distribuion within ptData}}

\end{DoxyCode}
